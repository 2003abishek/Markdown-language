# Markdown Resources

For more information on Markdown, check out these resources:

- [Markdown Guide](https://www.markdownguide.org/): A comprehensive guide to Markdown syntax and usage.
- [CommonMark](https://commonmark.org/): A reference implementation of Markdown.
- [Daring Fireball](https://daringfireball.net/projects/markdown/): The original Markdown specification by John Gruber.

With these resources, you can enhance your Markdown skills and explore its full potential.
